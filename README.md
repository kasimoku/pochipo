# pochipo

スマホで使える、単一HTMLのドット絵エディタ。描いたドット絵を Claude に渡せる **インデックス方式のJSON**（`// row N` コメント付き）として出力できます。

## 特長

- **キャンバス 3枚（C1 / C2 / C3）** をタブで切り替えて編集
- **カラーパレット 2セット（set1 / set2）** を切り替え。描き込み済みのピクセルは色を保持（焼き付け方式なのでセット切替で化けません）
- **カラーピッカー**（色相・彩度・明度のスライダー）で色を作りパレットに追加
- パレットのスウォッチは **タップで選択（スポイト）／ドラッグで並べ替え／長押し・右クリックで削除**
- **1段以上の undo / redo**
- **背景表示の切り替え**（市松 / 白 / グレー）
- **キャンバスサイズ可変**（1〜64）
- **JSON 入出力**: キャンバスJSON出力 / パレットJSON出力 / JSONインポート
- 状態は localStorage に自動保存

## 使い方

`pochipo.html` をブラウザで開くだけ（ビルド不要・依存なし）。スマホのブラウザでもそのまま動作します。

## 出力JSONの例

```json
{
  "palette": {
    "0": null,
    "1": "#23468C",
    "2": "#45A887"
  },
  "width": 3,
  "height": 2,
  "grid": [
    [1,2,0], // row 0
    [0,1,2]  // row 1
  ]
}
```

`0` は透明（背景）を表します。
